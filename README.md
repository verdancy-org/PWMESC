# PWMEsc

PWM ESC output module

## Required Hardware
pwm_motor_1, pwm_motor_2, pwm_motor_3, pwm_motor_4, pwm_motor_5, pwm_motor_6, ramfs

## Constructor Arguments
- `command_topic_name`: pwm_esc_cmd
- `esc_count`: 4
- `frequency`: 400
- `min_pulse_us`: 1000
- `max_pulse_us`: 2000
- `disarmed_pulse_us`: 1000
- `signal_timeout_ms`: 1000
- `task_stack_depth`: 1024

## Template Arguments
None
